# Exploring Swagger documentation

Exploring Swagger UI with OpenAPI example.

## Dependencies:
  - Node v8
  - Babel v7
  - Webpack v4


## Development

Start the webpack dev server and open localhost:8080 to swagger-ui with `npm run dev-server`
